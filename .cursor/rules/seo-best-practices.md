# 🚀 SEO Best Practices: Cursor Rules & Guidelines

## 1. Target High-Intent, Purchase-Ready Keywords
- Avoid broad, generic keywords (e.g., "running shoes").
- Focus on specific, high-intent keywords (e.g., "best running shoes for flat feet").
- Use keywords that signal buying intent and solve user needs.

## 2. Optimize Category Pages
- Use keyword-rich H1 and title tags.
- Add 500+ words of relevant, helpful content (buying guides, FAQs, etc.).
- Internally link to top-selling products.
- Implement schema markup for rich results.

## 3. Write Unique Product Descriptions
- Never copy-paste manufacturer descriptions.
- Write original, conversion-focused copy for each product.
- Add real customer reviews and photos.
- Optimize images with descriptive alt text.
- Use FAQ schema to answer common questions.

## 4. Internal Linking Strategy
- Structure links: Homepage → Categories → Products.
- Link blog posts to best-selling products.
- Link high-traffic pages to underrated product pages.
- Ensure every important page is linked from somewhere relevant.

## 5. Add Review Schema
- Implement review schema on product pages to show star ratings in search results.
- More stars = higher click-through rates and more sales.

## 6. Optimize Site Speed
- Compress images (preferably WebP format).
- Use a fast, lightweight theme.
- Implement CDN and caching for faster load times.

## 7. Track & Measure Everything
- Set up Google Analytics 4 to track traffic and conversions.
- Use Google Search Console to monitor rankings and fix issues.
- Add UTM parameters to track marketing channel performance.

## 8. Local SEO: Go All-In on Local Intent
- Target local, high-intent keywords (e.g., "Drain Cleaning in Austin").
- Build 'money pages' for each core service and location.
- Publish hyper-local content that answers local questions and builds trust.
- Optimize for 'near me' searches:
  - Embed Google Maps
  - Mention local landmarks and zip codes
  - Add "Serving [Nearby Cities]" in the footer
- Get main keywords in H1, H2, and URL slug.

## 9. Local Internal Linking
- Homepage links to top service pages.
- Service pages link to relevant blog posts.
- Blog posts link back to service pages.
- Avoid dead-end pages; ensure a strong internal link structure.

---

**Refer to these SEO rules for every e-commerce or local business project to maximize rankings, traffic, and conversions.**
